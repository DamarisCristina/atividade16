# Acampavera - Site Institucional

## Descrição

Este projeto foi desenvolvido utilizando HTML, CSS, JavaScript e W3.CSS com o objetivo de criar um site institucional para o Acampavera, um acampamento cristão voltado para jovens da região de Franca-SP.

O site apresenta informações sobre o evento, seus serviços, localização, parceiros, perguntas frequentes, formas de contato e redes sociais.

## Funcionalidades

- Menu lateral para navegação
- Menu responsivo para dispositivos móveis
- Slideshow automático de imagens
- Seção sobre o Acampavera
- Informações sobre localização
- Área de parceiros
- Perguntas frequentes (FAQ)
- Formulário de contato
- Link para redes sociais

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- W3.CSS
- Font Awesome
